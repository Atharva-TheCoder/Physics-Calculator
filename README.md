## Physics-Calculator
### Code by Atharva-TheCoder

## INSTALLATION [Termux] :

* `apt-get update -y`
* `apt-get upgrade -y`
* `pkg install python -y`
* `pkg install python2 -y`
* `pkg install git - y`
* `git clone https://github.com/Atharva-TheCoder/Physics-Calculator`

## RUN

* `cd Physics-Calculator`
* `ls`
* `python physics_calc.py`

## Update

* `cd $HOME && ls`
* `cd Physics-Calculator`
* `chmod +x *`
* `./update.sh`

OR

* `cd $HOME && ls`
* `cd Physics-Calculator`
* `bash update.sh`
